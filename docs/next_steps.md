# Next Steps for In-Memory Search Engine

## Verification Process
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Run unit tests:
   ```
   python -m unittest discover -s tests
   ```
3. Execute the main application:
   ```
   python src/main.py
   ```
